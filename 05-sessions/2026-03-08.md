## 워커 완료 자동 감지 — Stop Hook + Factory Detector

### 배경
워커가 작업 완료 후 `notify.py` 완료 명령을 실행하지 않으면 파이프라인이 멈추는 문제.
두 겹의 안전장치로 해결: Stop hook(즉시) + Factory detector(60초 안전망).

### 구현

1. **`scripts/worker_complete_hook.py`** — Stop hook (신규, ~90줄)
   - Claude Code turn 종료 시 발동 (글로벌 settings.json에 등록)
   - `CC_SESSION_NAME`으로 `cc-pool-N` 워커만 필터링
   - factory DB에서 활성 assignment 조회 → tmux scrollback 스캔
   - skip 패턴 5개 (AskUserQuestion, plan mode 등) 확인 후 `notify.py` 전송

2. **`scripts/feature_factory/completion_detector.py`** — 2차 안전망 (신규, ~100줄)
   - housekeep tick(60초)마다 실행
   - 활성 assignment의 tmux scrollback에서 idle 프롬프트(❯) + 2분 경과 감지
   - `dispatcher._handle_stage_complete()` 직접 호출
   - `_detected` set으로 중복 감지 방지 + 자동 정리

3. **`scripts/feature_factory/dispatcher.py`** — idempotency guard
   - `_handle_stage_complete()`에서 assignment 없으면 전체 early return
   - Stop hook + detector 양쪽 동시 발동 시 중복 처리 방지

4. **`scripts/feature-factory.py`** — `_housekeep()`에 detector 호출 추가

5. **`~/.claude/settings.json`** — Stop hook 등록 (timeout 10초)

### 테스트
- 기존 192 → 252 assertions (60개 추가)
- hook DB 조회, skip 패턴, idle 감지, timestamp 파싱, set 정리, idempotency guard
- 252/252 통과

### 아키텍처 포인트
- **이중 안전망**: B(Stop hook, 0초) + A(Factory detector, 최대 60초)
- **idempotency**: assignment 기반 guard — 먼저 도착한 쪽이 처리, 나중 쪽은 skip
- **비침습적**: 기존 코드 구조 변경 없이 새 모듈 2개 + 기존 함수 guard 추가

### 버그 수정
- `_is_idle()` — Claude Code tmux UI에서 마지막 줄이 statusbar(`⏵⏵ bypass permissions`)이라 `❯` 감지 실패
- 수정: 마지막 줄 → **하단 5줄 탐색**으로 변경 (statusbar, 구분선 감안)

## 승인 Revise 버튼 + 워커 액션 링크

### 배경
승인 알림에 approve/reject만 있어서 수정 요청이 불가능. 모바일에서 워커 접속 방법도 없음.

### URL Scheme 탐색
Claude 앱에서 워커 세션을 여는 deep link를 찾기 위해 다양한 URL 테스트:
- `claude://`, `anthropic://` — 등록된 URL scheme 없음
- `https://claude.ai/code` (경로 없음) — 브라우저로 리다이렉트
- `https://claude.ai/code?bridge=xxx` — 쿼리 파라미터라 Universal Link 불일치, 브라우저로 감
- **`https://claude.ai/code/{path}`** — 앱에 머물지만 세션 매핑 안 됨 (오류 표시)
- `apple-app-site-association` 확인: `/code/*`, `/share/*`, `/artifacts/*` 패턴 등록 (App ID: `Q6L2SF6YDW.com.anthropic.claude`)

**결론**: `https://claude.ai/code/open`으로 앱 Code 탭 진입 (오류 뜨지만 세션 목록 접근 가능) + `tmux attach -t cc-pool-N` 복사용 커맨드 제공. 완벽한 deep link는 Claude 앱 개선 대기.

1. **`notifier.py`** — revise 버튼 + 액션 링크
