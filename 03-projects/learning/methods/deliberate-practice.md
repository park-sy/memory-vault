# 의도적 연습 (Deliberate Practice)

## 개요

명확한 목표, 즉각적 피드백, 약점 집중 반복으로 전문성을 기르는 방법.
"1만 시간"이 아니라 "1만 시간의 의도적 연습"이 핵심.

## 원리

- 편안함 영역(comfort zone) 밖에서만 성장
- 즉각적 피드백이 없으면 잘못된 패턴이 고착됨
- 전체보다 약한 부분(sub-skill)에 집중해야 효율적
- 메타인지: 자신의 수행을 객관적으로 평가하는 능력 필요

## 절차

1. 구체적이고 측정 가능한 목표 설정
2. 현재 수준 대비 약간 어려운 과제 선택
3. 집중적 수행 (짧고 강도 높게)
4. 즉각적 피드백 확인
5. 약점 식별 → 해당 sub-skill 격리 연습
6. 반복

## 변형

- **AI 코치**: LLM이 수행 분석 + 피드백 제공 + 다음 과제 추천
- **마이크로 연습**: 15-20분 단위의 초집중 연습 세트
- **의도적 실패 연습**: 일부러 어려운 문제에 도전 → 실패 패턴에서 학습

## When to Use

- 기술(skill) 향상이 목표일 때 (코딩, 글쓰기, 분석 등)
- 정체기(plateau)를 돌파하고 싶을 때
- 피드백을 빠르게 받을 수 있는 환경

## When NOT to Use

- 개념 이해 단계 (이해 없이 연습하면 잘못된 패턴 강화)
- 피드백이 불가능한 영역 (효과 판별 불가)
- 지나친 피로 상태 (연습 품질 저하)
