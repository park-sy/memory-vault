# 교차 학습 (Interleaving)

## 개요

한 주제를 집중적으로 반복(blocking)하는 대신, 여러 주제/유형을 섞어서 학습하는 방법.
단기 성과는 낮아 보이지만 장기적 전이력이 높다.

## 원리

- 판별 학습(discrimination learning): 서로 다른 것을 비교하며 차이를 파악
- 검색 전략 다양화: 매번 다른 맥락에서 인출 → 유연한 검색 경로 형성
- 주의 환기 효과: 전환 자체가 집중력 유지에 도움

## 절차

1. 학습할 주제/유형 3개 이상 준비
2. 한 주제를 15-20분 학습
3. 다른 주제로 전환 (무작위 또는 순환)
4. 각 주제 간 유사점/차이점 의식적으로 비교
5. 세션 후 각 주제의 핵심 요약

## 변형

- **교과 교차**: 완전히 다른 분야를 번갈아 학습
- **유형 교차**: 같은 분야 내 다른 문제 유형을 섞음
- **AI 셔플**: LLM이 학습 순서를 랜덤화 + 주제 간 연결점 제시

## When to Use

- 유사하지만 다른 개념을 구별해야 할 때
- 장기 기억과 전이가 중요할 때
- [[spaced-repetition]]과 결합 시 효과 증폭

## When NOT to Use

- 완전히 새로운 주제의 초기 학습 (기본 이해부터 필요)
- 시간 압박이 심할 때 (단기 성과가 더 중요한 경우)
