# 소크라테스식 문답 (Socratic Method)

## 개요

답을 직접 알려주는 대신, 연쇄적 질문으로 학습자가 스스로 답에 도달하게 하는 방법.
AI와의 대화에서 특히 효과적 — AI가 일관된 질문 체인을 유지할 수 있음.

## 원리

- 인지적 갈등(cognitive conflict): 질문이 기존 믿음과 충돌하면 깊은 사고 유발
- 자기 발견 효과: 스스로 도달한 결론이 전달받은 결론보다 기억/적용에 강함
- 메타인지 훈련: "왜 그렇게 생각하는가?"가 사고 과정 자체를 의식하게 함

## 절차

1. 학습 주제 선정
2. AI에게 "소크라테스식으로 질문해줘, 답을 바로 알려주지 마" 요청
3. AI 질문 → 학습자 답변 → AI 후속 질문
4. 모순/비일관성 발견 시 AI가 지적
5. 학습자가 스스로 결론에 도달
6. AI가 정리 + 보충

## 변형

- **역소크라테스**: 학습자가 AI에게 질문 → AI가 "틀린 답" → 학습자가 반박
- **점진적 힌트**: 막히면 직접 답 대신 한 단계 힌트
- **[[contextual-analogy]] 결합**: 질문에 상엽 경험 기반 비유 포함

## When to Use

- 논리적 추론 능력을 기르고 싶을 때
- 기존 이해의 정확성을 검증하고 싶을 때
- 암기보다 깊은 이해가 목적일 때

## When NOT to Use

- 시간 압박 (소크라테스식은 느림)
- 기초 지식이 전혀 없는 상태 (질문에 답할 수 없음)
- 팩트/절차 암기 목적
