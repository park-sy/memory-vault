# 간격 반복 (Spaced Repetition)

## 개요

시간 간격을 두고 정보를 반복 복습하여 장기기억 전이를 극대화하는 방법.
에빙하우스의 망각 곡선에 기반. 기억이 희미해질 때 복습하면 다음 망각까지의 간격이 늘어난다.

## 원리

- 망각 곡선: 학습 직후 급격히 망각, 시간이 지나며 완만해짐
- 간격 효과(spacing effect): 집중 반복보다 분산 반복이 장기기억에 효과적
- 검색 강도(retrieval strength): 어려울수록 기억이 강화됨

## 절차

1. 학습 자료를 카드 단위로 분할
2. 첫 복습: 1일 후
3. 정답 시 간격 증가 (1일 → 3일 → 7일 → 14일 → 30일 ...)
4. 오답 시 간격 초기화
5. 난이도별 간격 조정

## 변형

- **Anki 알고리즘**: SM-2 기반 자동 간격 조정
- **Leitner 시스템**: 박스 5개로 물리적 분류
- **AI 적응형**: LLM이 이해도 판단하여 간격/질문 난이도 조정

## When to Use

- 팩트, 용어, 공식 등 **선언적 지식** 습득
- 어휘 암기 (언어 학습)
- 자격증/시험 준비

## When NOT to Use

- 개념 이해/응용이 목적인 경우 (이해 없이 암기만 강화됨)
- 절차적 지식 (코딩, 운동 등 — 직접 수행이 더 효과적)
- 창의적 문제 해결
