# 사례 분석 (Case Study Analysis)

## 개요

실제 사례를 분석하여 판단력과 의사결정 능력을 기르는 방법.
하버드 비즈니스 스쿨 방식에서 유래. 이론을 현실 맥락에서 적용하는 훈련.

## 원리

- 상황적 학습(situated learning): 실제 맥락에서 배운 지식이 전이 잘 됨
- 다중 관점: 같은 사례를 다른 각도에서 분석하며 사고 유연성 증가
- 패턴 인식: 여러 사례에서 공통 패턴을 추출하면 새 상황에 적용 가능

## 절차

1. 사례 선택 (실제 사건, 성공/실패 모두)
2. 배경, 상황, 제약 조건 정리
3. 핵심 의사결정 포인트 식별
4. "나라면 어떻게 했을까?" 먼저 작성
5. 실제 결과와 비교 분석
6. 추출한 교훈/패턴 기록

## 변형

- **역사례 분석**: "왜 실패했나?"에 집중 → 회피 패턴 학습
- **비교 사례**: 유사 상황, 다른 결과 → 핵심 변수 식별
- **AI 토론**: LLM과 사례에 대해 다른 관점으로 토론

## When to Use

- 의사결정 능력 훈련
- 새 분야 진입 시 빠른 맥락 파악
- [[mental-model-building]]의 입력 재료로 활용

## When NOT to Use

- 추상적/이론적 개념 학습 (사례가 특수해서 일반화 위험)
- 기초 지식이 없는 상태 (사례 분석이 불가)
