# 파인만 기법 (Feynman Technique)

## 개요

복잡한 개념을 간단한 말로 설명할 수 있을 때까지 반복하는 방법.
"설명할 수 없으면 이해한 게 아니다."

## 원리

- 설명 생성 효과(generation effect): 자신의 말로 재구성하면 깊은 처리 발생
- 지식 격차 발견: 설명 과정에서 이해가 부족한 지점이 드러남
- 단순화 강제: 전문 용어 없이 설명하면 핵심만 남음

## 절차

1. 주제 선택
2. 12세 아이에게 설명하듯 간단한 말로 작성
3. 막히는 부분 → 원자료로 돌아가서 재학습
4. 비유, 시각화로 더 단순하게 다듬기
5. 2-4 반복

## 변형

- **AI 파인만**: LLM에게 "내가 설명할 테니 틀린 부분 짚어줘" 요청
- **역파인만**: LLM이 설명 → 학습자가 틀린 점/부족한 점 찾기
- **[[contextual-analogy]] 결합**: 일반 비유 대신 상엽 경험 기반 비유로 설명

## When to Use

- 새 개념의 깊은 이해가 필요할 때
- "알 것 같은데 설명은 못하겠는" 상태
- 복잡한 시스템/이론을 정리할 때

## When NOT to Use

- 단순 암기 (간격 반복이 더 효율적)
- 이미 잘 아는 주제 (시간 낭비)
