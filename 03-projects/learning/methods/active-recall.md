# 능동 회상 (Active Recall)

## 개요

정보를 수동적으로 다시 읽는 대신, 기억에서 능동적으로 꺼내는 연습.
"읽기" 대신 "테스트"가 학습의 핵심이라는 검증된 방법.

## 원리

- 테스트 효과(testing effect): 정보를 인출하는 행위 자체가 기억을 강화
- 바람직한 어려움(desirable difficulty): 쉽게 꺼내지 못할수록 기억 강화 효과 큼
- 메타인지 보정: 자신의 이해도를 정확히 파악 가능

## 절차

1. 자료를 한 번 학습
2. 자료를 덮고 핵심 내용을 스스로 떠올림
3. 떠올리지 못한 부분 확인
4. 약한 부분에 집중하여 반복

## 변형

- **플래시카드**: 질문-답변 카드 기반 인출 연습
- **자유 회상(free recall)**: 주제에 대해 아는 것을 전부 적기
- **정교화 질문(elaborative interrogation)**: "왜 그런가?" 질문으로 인출 + 이해
- **AI 문답**: LLM이 질문 생성 → 학습자 답변 → LLM 피드백

## When to Use

- 시험 준비, 개념 확인
- [[spaced-repetition]]과 결합 시 최대 효과
- 자기 이해도를 정확히 파악하고 싶을 때

## When NOT to Use

- 최초 노출 단계 (기본 이해 없이 인출 불가)
- 절차적 스킬 학습 (직접 수행이 더 효과적)
