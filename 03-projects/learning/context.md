# Learning — 도메인 컨텍스트

AI를 활용한 학습 최적화 도메인팀. 코어팀에 전달할 최소 컨텍스트.

## 프로젝트 요약

다양한 학습 방법을 탐색 → 상엽에게 적용/실험 → 데이터 기반으로 최적 방법 수렴.
소프트웨어 프로젝트가 아닌 **실험/최적화** 유형 도메인팀.

## 학습 프로필 — 상엽

### 강점
- 비유를 통한 이해가 빠름 (특히 자기 경험에 기반한 비유)
- 시스템/구조적 사고 (개발자 배경)
- "왜?"에 대한 강한 호기심 — 원리를 이해해야 적용 가능
- 반복보다 패턴 추출을 선호

### 약점
- 단순 암기형 학습에 약함
- 흥미가 없으면 집중도 급락
- 초기 진입 장벽이 높은 주제에 대한 인내심 낮음

### 선호 모달리티
- 대화형 > 독서형
- 시각화 > 텍스트
- 실습 > 이론 선행
- 짧은 반복 > 긴 세션

## 핵심 가설

> **개인 경험 기반 비유(Contextual Analogy) + AI 선순환 구조**가 가장 효과적인 학습 방법이다.

memory-vault에 상엽의 경험이 축적됨 → AI가 그 경험을 기반으로 비유 생성 → 이해도 상승 → 새 경험 축적 → 비유 정확도 상승 (선순환).

이 가설은 아직 **실험 전**이며, 체계적 검증이 필요하다.
